## 플랫폼별 핵심 규칙

### Threads
- 목적: 팔로워 유입 → 프로필 링크 → 스토어
- 형식: 숫자 리스트 5-10개, "[주제] N가지" 제목
- **제목: 28자 이하** (공백 포함, 모바일 가독성)
- 톤: 담백하고 절제된
- 금지: 이모지, 해시태그, 상품 언급, 판매 유도

### Instagram
- 목적: 상품 직접 홍보
- 형식: 카드뉴스 5-7장 + 릴스 스크립트
- 톤: 친근하고 따뜻한
- 필수: 해시태그 (#일본구매대행 #일본직구 #드리치시)

### 네이버 블로그
- 목적: 검색 유입 + 상품 상세 정보
- 형식: SEO 최적화 리뷰, 1500자 이상
- 톤: 정보 전달 관점, 신뢰감 있는
- 필수: 스토어 링크 2-3회 자연 삽입

## 학습 로그

### Threads 성공 패턴
- 반응 좋은 숫자: 7, 8, 9, 10 (5개 미만은 약함)
- 반응 좋은 주제 키워드: "30대", "품격", "부자", "고급스러운", "끊은", "배운", "취향"
- 마무리: 질문형("스친들은?")이 댓글 유도에 효과적
- 여운형 마무리도 저장 유도에 좋음
- **공간/인테리어 주제가 빠른 확산에 효과적** (8분에 1.1천 조회)
- 고급스러운/부자 주제가 빠른 확산에 효과적

### 검증된 Threads 제목 패턴
```
- [N]살이 되고 배운/깨달은 N가지
- [형용사]한 사람들의 N가지 특징
- 진짜 [대상]의 N가지
- [대상]에 꼭 있는 것들 N가지
- [시기]에 끊은/시작한 N가지
- [형용사] [대상]에 꼭 있는 N가지  ← NEW (바이럴 검증)
```

### S+ 등급 성공 분석 (돈 많아 보이는 집 - 1.1만 조회)
**핵심 성공 요인**:
- **완벽한 주체**: "돈 많아 보이는 집" → 호기심 + 동경 + 구체성 극대화
- **시각적 아이템**: 신발 두세 켤레, 사이드 테이블, 호텔 수건 등 상상 가능
- **반전 마무리**: "돈 많아 보이는 건 물건이 많아서가 아니다" → 저장 유도
- **댓글 반응**: "우리집하고는 거리가 멀어", "100프로 똑같네", "진짜 부자가 되보자"
- **시간당 조회 458회** (역대 최고)

### S 등급 성공 패턴 분석
**"부자들이 돈 아끼지 않는 8가지" (4.3천, 리포 10)**
- 리포스트 최다 기록 → **공유하고 싶은 인사이트**
- 항목: 침구류, 의자, 신발, 식사 자리, 건강 검진, 선물, 여행, 배움
- **"싼 게 비싼 거였던 적이 한두 번이 아니다"** → 명언형 마무리

**"향이 좋은 부자집들의 공통점" (3.9천)**
- 후각 감각 자극 → 차별화된 콘텐츠
- 상품 연관 가능성 높음 (디퓨저)

**"진짜 부자들의 아침 루틴" (2.8천, 댓글 9)**
- 댓글: "일어나자마자 신문 4개 읽는데 부자되긴 글렀군요"
- 루틴 주제지만 **"진짜 부자들"** 주체가 강력해서 성공

**"월급 300으로 여유롭게 사는" (3.2천)**
- **공감형 주체**: 대다수 직장인 타겟
- "다 하는것들이다" 댓글 → 실천 가능성 높은 항목

**"일 잘하는 직장인 가방 속" (2.5천, 댓글 3)**
- 구체적 아이템 리스트: 보조배터리, 노트, 볼펜, 명함, 핸드크림
- 댓글: "B2B 영업 10년 하면서 6번이 제일 중요하더라"

**"35살에 정리한 관계" (1.4천, 좋아요 32)**
- **감정 주제**: 관계 정리 → 공감 유도
- 좋아요 대비 조회 비율 높음 → 저장 많이 됨

### B등급 분석 - 개선 포인트
**"품격 있는 사람들의 공통점" (630조회, 좋아요 25)**
- 좋아요는 높지만 조회수 낮음 → **알고리즘 노출 실패**
- 주체가 추상적 ("품격")
- 댓글: "조용히 말한다(말소리 크기가 아님)"

**"자산 10억 부자들이 안 사는 것" (720조회)**
- 주체는 S티어지만 **"안 하는" 부정형이 덜 매력적**
- "~에 꼭 있는" > "안 하는/안 사는"

### C등급 실패 분석 - 피해야 할 패턴
**"자취 5년 차 방" (273조회)**
- 문제: 타겟층이 너무 좁음 (자취생 한정)
- 개선: "혼자 사는 사람" 또는 "독립한 30대"로 확장

**"월 1000만원 버는 사람들의 저녁 습관" (345조회)**
- 문제: 같은 주체(월 1천)인데 "진짜 부자 아침 루틴"(2.8천)보다 저조
- 원인: **저녁 습관 < 아침 습관** (동경심 차이)

**"30대 후반이 되고 깨달은" (46조회) - 역대 최저**
- 문제: 항목이 너무 추상적 ("관계는 깊게 유지", "건강 챙겨야")
- 원인: 설교 톤 + 뻔한 내용
- 교훈: **깨달음/조언형 < 아이템/물건형**

### 피해야 할 패턴 (2026-05-02 업데이트)
- 15개 이상 긴 리스트: 스크롤 이탈
- 추상적 주제: 구체적 숫자/사례가 더 효과적
- 설교 톤: 공감/관찰 톤이 더 효과적
- **추상적 행동/습관 리스트**: 구체적 물건/아이템보다 반응 저조
- **"주말/저녁 루틴"**: 아침 루틴보다 동경심 낮음
- **"~이 되고 깨달은"**: 설교 톤 느껴짐, 46조회 역대 최저 기록
- **타겟층 좁은 주체**: "자취 5년 차" (273조회) vs "혼자 사는 30대"
- **부정형 제목**: "안 하는/안 사는" < "꼭 있는/하는"
