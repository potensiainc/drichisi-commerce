**1단계: 히스토리 파일 읽기**

## SEO 전략

### 제목 최적화 - 바이럴 클릭 유도형

**핵심 원칙**: 검색에 노출되어도 클릭하고 싶은 제목이어야 한다. 단순 키워드 나열이 아닌, 호기심과 사회적 증거를 자극하는 제목.

**제목 공식 (필수 선택):**

| 패턴 | 공식 | 예시 |
|------|------|------|
| **타겟 화제형** | 요즘 [타겟]에서 난리 났다는 [상품명] | 요즘 30대 직장인 사이에서 난리 났다는 란도린 디퓨저 |
| **여행 필수형** | 일본 가면 무조건 사야 되는 [상품 카테고리] | 일본 가면 무조건 사야 되는 디퓨저 방향제 |
| **선물 화제형** | 집들이 선물로 난리 난 [상품명] | 집들이 선물로 난리 난 란도린 디퓨저 |
| **중독형** | 한번 쓰면 계속 쓸 수밖에 없다는 [상품명] | 한번 쓰면 계속 쓸 수밖에 없다는 란도린 디퓨저 |
| **비교 우위형** | [일반 상품] 말고 이거 쓰세요, [상품명] | 다이소 방향제 말고 이거 쓰세요, 란도린 디퓨저 |
| **후회형** | 진작 살 걸 후회한 [상품명] 리뷰 | 진작 살 걸 후회한 란도린 디퓨저 리뷰 |
| **발견형** | 일본에서 찾은 [형용사] [상품명] | 일본에서 찾은 향수병 같은 디퓨저 |
| **입소문형** | SNS에서 핫한 [상품명], 직접 써봤습니다 | SNS에서 핫한 란도린 디퓨저, 직접 써봤습니다 |


**금지 패턴 (클릭률 낮음):**

**제목 검증 체크리스트:**
- [ ] 검색 결과에서 이 제목 보면 클릭하고 싶은가?
- [ ] 사회적 증거 포함? (난리, 핫한, 인기 등)

## 글쓰기 가이드

### 톤앤매너 - 실제 사용자 리뷰처럼

**핵심 원칙**: 광고 티 나면 안 됨. 진짜 구매해서 써본 사람이 친구한테 얘기하듯 작성.

**말투 규칙:**
```
✅ 사용: ~했는데요, ~했어요, ~그랬나봐요, ~인 것 같아요, ~거든요
❌ 금지: ~입니다, ~합니다, ~드립니다 (다나까체 절대 금지)
```

**예시 변환:**
| 다나까체 (금지) | 친근체 (사용) |
|----------------|--------------|
| 구매했습니다 | 구매했어요 / 샀는데요 |
| 추천드립니다 | 추천해요 / 좋은 것 같아요 |
| 사용해보았습니다 | 써봤는데요 / 써봤거든요 |
| 배송되었습니다 | 배송 왔는데요 / 도착했어요 |
| ~인 것 같습니다 | ~인 것 같아요 / ~그랬나봐요 |
| 안내드립니다 | 알려드릴게요 / 말씀드릴게요 |

**실제 리뷰처럼 쓰는 법:**
```
❌ "이 제품은 [특징]이 있어 [상황]에 적합합니다."
✅ "저도 처음엔 반신반의했는데요, 써보니까 진짜 다르더라고요."

❌ "드리치시에서 일본 정품을 구매대행하고 있습니다."
✅ "여기서 구매했는데요, 일본 정품이라 안심이에요."

❌ "[이런 분]들께 추천드립니다."
✅ "저처럼 [이런 분]들한테 딱인 것 같아요."

❌ "현관에 배치하면 인테리어 효과가 있습니다."
✅ "현관에 뒀는데 진짜 분위기 달라졌어요."
```

**자연스러운 표현 예시:**
- "친구 집들이 선물로 줬는데요, 반응이 대박이었어요"
- "처음엔 좀 비싸다 싶었는데, 쓰다 보니까 이해가 되더라고요"
- "사실 저도 인스타에서 보고 혹해서 샀거든요"
- "한 달 정도 썼는데 아직도 향이 잘 나와요"
- "솔직히 처음엔 기대 안 했는데, 의외로 좋았어요"

**DO:**
- 1인칭 시점으로 작성 (나, 저, 제가)
- 구매/사용 경험담처럼 작성
- 솔직한 장단점 언급 (단점도 살짝)
- 구어체, 줄임말 자연스럽게 사용
- 감탄사, 구어적 표현 OK (진짜, 대박, 완전 등)

**DON'T:**
- 다나까체 (~입니다, ~합니다) 절대 금지
- 판매자 입장 표현 금지
- 너무 완벽한 칭찬만 (의심받음)
- 딱딱한 설명조

5. **링크**: URL 전체를 그대로 표기 (네이버가 자동 링크 처리)
6. **강조**: 별도 마크업 없이 문맥으로 강조

## 이모지 사용 규칙 (필수)

### Blogspot/HTML 출력 시 이모지 절대 금지

**문제**: 이모지는 Blogspot 등 일부 플랫폼에서 HTML 엔티티 코드(`&#x...;`)로 깨져서 표시됨.

**해결책**: HTML 출력 시 이모지 대신 텍스트/CSS 스타일 사용

| 이모지 (금지) | 대체 텍스트 |
|--------------|------------|
| 🇯🇵 | JP |
| 👍 | + (CSS 아이콘) |
| 🤔 | - (CSS 아이콘) |
| 📷 | [이미지] |
| 💡 | TIP \| |
| 🎀 | POINT \| |
| ✅ | + |
| ❌ | - |
| 👉 | → |
| 📦 | PRODUCT INFO |
| ⚖️ | (섹션 번호 01, 02...) |
| 💝 | (섹션 번호 01, 02...) |

### 네이버 블로그 (플레인 텍스트) 출력

**허용**: 📷 같은 기본 이모지만 제한적 사용 (이미지 위치 표시용)
**주의**: 국기, 복합 이모지 등은 깨질 수 있으므로 자제

### HTML 아이콘 대체 패턴

```html
<!-- 장점/단점 아이콘 -->
<span class="pros-icon">+</span>  <!-- 👍 대체 -->
<span class="cons-icon">-</span>  <!-- 🤔 대체 -->

<!-- 섹션 제목 아이콘 -->
<span class="section-title-icon">01</span>  <!-- 이모지 대체, 숫자 사용 -->

<!-- 배지 -->
<span class="hero-badge">JP 일본 직구 리뷰</span>  <!-- 🇯🇵 대체 -->

<!-- CTA 버튼 -->
상품 바로가기 &rarr;  <!-- 👉 대체, HTML 엔티티 사용 -->
```

## Blogspot HTML 출력 가이드

### 요청 시 HTML 버전 생성

사용자가 "blogspot용", "html로" 등을 요청하면 HTML 버전 추가 생성.

### HTML 출력 규칙

1. **이모지 금지**: 위 대체 텍스트/CSS 패턴 사용
2. **실제 이미지 URL 사용**: `product_info.json`의 `variant_images` 또는 `all_image_urls`에서 가져옴
3. **인라인 CSS**: 외부 스타일시트 금지
4. **Google Fonts**: Noto Sans KR 사용 가능
5. **반응형**: max-width: 680px 권장

### 이미지 URL 가져오기

```
1. products/[상품명]/product_info.json 읽기
2. variant_images.default 배열에서 고해상도 URL 선택
3. _AC_SL1500_ 포함된 URL 우선 사용
4. HTML img src에 직접 삽입
```

### 출력 저장 위치

```
네이버 블로그: marketing/naver-blog/[상품명]-리뷰.txt
Blogspot HTML: marketing/naver-blog/[상품명]-blogspot.html
```
