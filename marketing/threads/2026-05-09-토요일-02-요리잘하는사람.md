# 요리 잘하는 사람들 주방 공통점 8가지

**예상 등급**: S

요리 잘하는 사람들 주방 공통점 8가지

1. 칼이 잘 든다
2. 냄비가 3개 이하다
3. 양념장이 미리 되어 있다
4. 냉장고가 비어 있다
5. 자주 쓰는 건 손 닿는 곳에 있다
6. 후라이팬 상태가 좋다
7. 계량컵을 쓴다
8. 싱크대가 항상 비어 있다

요리는 실력보다 환경이다.
