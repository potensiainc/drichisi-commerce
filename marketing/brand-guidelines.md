## Threads 콘텐츠 스타일 가이드

### 성공 패턴
- 숫자 리스트형 (5-10개 항목)
- 제목: "[주제] N가지" 형식
- 간결한 문장 (한 줄에 하나의 포인트)
- 이모지 절대 사용 안함
- 마지막에 질문 또는 여운 있는 마무리
- 상품/스토어 언급 절대 금지

### 주제 카테고리
1. 품격/고급스러움
2. 30대 라이프스타일
3. 돈/소비 철학
4. 언어/태도
5. 취향/라이프스타일

### 실제 성공 예시

```
37살 아재가 즐겨 입는 옷 브랜드
1. Theory - 대학생 때부터 최애. 결국 또 삼
2. Time - 국내 브랜드 중 이만한 핏 찾기 힘듦
3. 폴로 - 클래식은 질리지가 않음
4. COS - 미니멀 기본템은 여기
5. 무신사 스탠다드 - 가성비 헤비 맨투맨 좋음
취향이 쉽게 변하지 않네.
스친들은 어떤 브랜드 좋아해?
```

```

1. 비싼 게 아니라, 오래 쓸 걸 산다.

```
